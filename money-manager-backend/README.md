# Money Management App 💰

A full-stack personal finance management application that helps users track their income, expenses, and overall financial activity securely.

## Features
- User authentication with JWT
- Add income and expense transactions
- View all transactions
- Update or delete transactions
- Get transaction summaries (income, expenses, balance)
- Protected API routes with authentication middleware

## Tech Stack
**Backend:** Node.js, Express.js, MongoDB  
**Authentication:** JWT  
**Tools:** Postman, Git, GitHub

## Purpose
Built to practice backend development concepts such as REST APIs, authentication, middleware, database operations, and secure route handling.
